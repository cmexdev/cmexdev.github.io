how to make your own website

things you'll need

  • a computer (Windows, Mac, Linux)
  • a GitHub account
  • GitHub desktop
  • a code editor (Visual Studio Code recommended)
  • knowledge with websites and computers

please consider that this documentation is not official and will be very brief

create a folder that will hold the html files for your website. then create a new repository in GitHub desktop and title it with yourusername.github.io and select the folder you made earlier. for example i would create a new repository called piccolowen.github.io (which is how this website is hosted). once done select the folder where you are planning to place your html files, and right click and click open with Code, which will then open that folder in Visual Studio Code. In VSCode, create a new file and title it index.html a name other than indexwill not work because GitHub looks for index only. in index.html, you can type your html code. after you have coded a bit, go to GitHub desktop and commit the new file changes then publish them to GitHub.com. you site should automatically be published to yourusername.github.io!
