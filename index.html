My life

I have been a musician for the majority of my life and have been constantly improving my skills in music production and performing.

I started coding simple HTML files on my family's Mac, that's when I started my web development. When I look back at those times, I am amazed at my lack of knowledge.

Once I got my own computer (a cheap laptop that I still use), my knowledge shot up. I started using C++, some C# (Unity), Visual Basic, and still coding a small amount of HTML.

I was using C++ for a while, almost forgetting that HTML existed. (at this point a barely used JavaScript). I had always wanted a website, so one day I tried some things then I figured GitHub has a free way to host a site.

This is basically where we are today. Oh wait, I use JavaScript a lot now. I used to hate JavaScript, but now I really like it.
